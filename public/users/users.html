How to use the Search

Put in the exact name or a Spotify User ID (Recommended) to fetch all information

Why can't I find myself?

Spotify's API does not allow searching Users, however you can simply put in your Spotify User ID to get all data! You can search up your Display Name, but it not always works, due to how Spotify's ID fetching work. We suggest using User IDs, found in the URL of a User page - that always works
